# 🏭 Daikibo Industrial Predictive Maintenance & Telemetry Analytics

### 📌 Project Overview
This project focuses on analyzing large-scale telemetry and IoT sensor data from **Daikibo**, a global manufacturing company with automated production lines across 4 major factories:
* **Daikibo Factory Meiyo** (Tokyo, Japan)
* **Daikibo Factory Seiko** (Osaka, Japan)
* **Daikibo Berlin** (Berlin, Germany)
* **Daikibo Shenzhen** (Shenzhen, China)

The primary challenge was handling massive data streams to identify patterns of unplanned downtimes and machine breakdowns, enabling the maintenance teams to take proactive measures.
